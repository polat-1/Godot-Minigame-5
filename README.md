# Godot Minigame 5
## Devlog
My cone count UI works by having a cone count variable, visually represented by a text UI called "ConesText". A function in every cone detects whether or not a vehicle has hit it. If it has been hit, the cone activates a function which will increase the cone count varaible by 1, and updates the ConesText UI to show the updated number.
